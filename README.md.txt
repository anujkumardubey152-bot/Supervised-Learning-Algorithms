# Supervised Learning Algorithms using Scikit-learn

## Project Overview

This project demonstrates the implementation of three fundamental supervised learning algorithms using Python and Scikit-learn.

The project covers both regression and classification tasks.

Algorithms implemented:

- Linear Regression
- Logistic Regression
- Decision Tree Classifier

---

## Technologies Used

- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
